Best Splash Help you to Display graphical images via splash before full page load.
 
== Description ==
 
Keep any WordPress site look cool, attractive, and engage your readers.

"First impression is last impression"

A splash page can be a welcome screen to the main website or a teaser that gets visitors excited for the website they’re about to view.

In Many cases websites breaks html dua to heavy css/js files and looks ugly which makes user annoy and this plugin will help to look website good. Because many sites take time to load and this effect will help you to load animation untill full page load.  

 
== Installation ==

1. Install as regular WordPress plugin.

2. Go your Pluings setting via WordPress Dashboard and activate it.

3. After activating you will see a menu on left side and go to "Best Splash” 
setting page and change your setting and save it.

4. Enjoy!!!
